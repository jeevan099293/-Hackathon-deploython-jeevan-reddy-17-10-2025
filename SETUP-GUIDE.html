Step 3: Set Up Git Repository

Open Command Prompt in your portfolio folder and run:

git add .
git commit -m "Initial commit: Personal portfolio website"

Step 4: Create GitHub Repository

  1. Go to GitHub.com/new
  2. Create a new repository (e.g., "portfolio" or "personal-website")
  3. Keep it PUBLIC (required for GitHub Pages)
  4. Don't initialize with README

Step 5: Push to GitHub

Replace YOUR-USERNAME and YOUR-REPO with your actual GitHub details:

git remote add origin https://github.com/YOUR-USERNAME/YOUR-REPO.git
git branch -M main
git push -u origin main
